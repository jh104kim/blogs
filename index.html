JH의 AI 뿌시기

막막한 프롬프트 한 줄이 업무를 바꾸는 실질적인 자동화의 시작이 되고, 수많은 에이전트와의 문답 끝에 비로소 AI를 도구 삼아 비즈니스를 설계한다.

이 저장소는 Next.js 정적 보내기 기반 블로그 소스입니다. 글은 content/의 마크다운으로 관리됩니다.

GitHub에서 소스 보기 포트폴리오 (김정현)

로컬에서 블로그 실행

  • git clone https://github.com/jh104kim/blogs.git
  • cd blogs 후 npm install
  • 개발: npm run dev → http://localhost:3000
  • 정적 빌드: npm run build → 산출물은 out/ 폴더

GitHub Pages로 전체 사이트 올리기

이 페이지(index.html)는 랜딩용입니다. 글 목록·포스트까지 같은 도메인에서 보이게 하려면 저장소 Settings → Pages에서 배포 브랜치를 지정한 뒤, out/ 빌드 결과를 해당 브랜치(예: gh-pages) 루트에 푸시하거나 Actions로 배포하면 됩니다.
